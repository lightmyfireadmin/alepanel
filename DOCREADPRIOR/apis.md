# APIs & External Services - Alecia Panel

## 1. Neon DB
- **Role:** Primary serverless PostgreSQL database.
- **Usage:** All relational data storage.

## 2. Microsoft Graph
- **Role:** Productivity integration.
- **Usage:** Accessing Outlook emails, calendar events, and OneDrive files related to M&A mandates.

## 3. Mistral / Groq
- **Role:** AI analysis.
- **Usage:** Processing company descriptions, matching buyers to sellers, and summarizing meeting notes.

## 4. Resend
- **Role:** Transactional email.
- **Usage:** Internal alerts for new leads or operation status changes.

## 5. Vercel Blob
- **Role:** Object storage.
- **Usage:** Secure storage for Pitch Decks, LoIs (Letters of Intent), and NDAs.
